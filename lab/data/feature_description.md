# German Credit 数据集特征说明

## 数据集概述

German Credit 数据集是德国信用风险数据集，包含1000个样本，用于评估客户的信用风险。数据集共有21个特征，其中20个属性特征和1个目标变量（信用评分结果）。

## 目标变量

| 特征名 | 说明 | 取值 |
|--------|------|------|
| class | 信用评分结果 | 1 = 好客户（低风险），2 = 坏客户（高风险） |

## 属性特征说明

### 1. checking_status（支票账户状态）
- **类型**: 类别型
- **说明**: 现有支票账户的状态
- **取值**:
  - A11: < 0 DM
  - A12: 0 <= ... < 200 DM
  - A13: >= 200 DM / 薪资分配至少1年
  - A14: 无支票账户

### 2. duration（贷款期限）
- **类型**: 数值型
- **说明**: 贷款期限（月）
- **取值范围**: 4 - 72个月

### 3. credit_history（信用历史）
- **类型**: 类别型
- **说明**: 客户的信用历史记录
- **取值**:
  - A30: 没有贷款记录/所有贷款已还清
  - A31: 所有贷款已按时还清
  - A32: 现有贷款按时还款至今
  - A33: 存在延迟还款记录
  - A34: 存在关键账户/其他银行存在风险

### 4. purpose（贷款目的）
- **类型**: 类别型
- **说明**: 贷款的用途
- **取值**:
  - A40: 购买新车
  - A41: 购买二手车
  - A42: 购买家具/设备
  - A43: 购买收音机/电视
  - A44: 购买家用电器
  - A45: 维修
  - A46: 教育
  - A47: 度假
  - A48: 再培训
  - A49: 商业用途
  - A410: 其他

### 5. credit_amount（贷款金额）
- **类型**: 数值型
- **说明**: 贷款金额（德国马克 DM）
- **取值范围**: 250 - 18424 DM

### 6. savings_status（储蓄账户状态）
- **类型**: 类别型
- **说明**: 储蓄账户/债券的状态
- **取值**:
  - A61: < 100 DM
  - A62: 100 <= ... < 500 DM
  - A63: 500 <= ... < 1000 DM
  - A64: >= 1000 DM
  - A65: 无储蓄账户/未知

### 7. employment（就业状态）
- **类型**: 类别型
- **说明**: 当前就业状态
- **取值**:
  - A71: 失业
  - A72: 就业 < 1年
  - A73: 就业 1 <= ... < 4年
  - A74: 就业 4 <= ... < 7年
  - A75: 就业 >= 7年

### 8. installment_commitment（分期付款承诺）
- **类型**: 数值型
- **说明**: 分期付款占可支配收入的百分比
- **取值范围**: 1 - 4（代表不同百分比区间）

### 9. personal_status（个人状态）
- **类型**: 类别型
- **说明**: 个人状态和性别
- **取值**:
  - A91: 男性 - 离婚/分居
  - A92: 女性 - 离婚/分居/已婚
  - A93: 男性 - 单身
  - A94: 男性 - 已婚/丧偶
  - A95: 女性 - 单身

### 10. other_parties（其他担保人）
- **类型**: 类别型
- **说明**: 其他债务人/担保人
- **取值**:
  - A101: 无
  - A102: 共同申请人
  - A103: 担保人

### 11. residence_since（居住时间）
- **类型**: 数值型
- **说明**: 在当前住所居住的年数
- **取值范围**: 1 - 4年

### 12. property_magnitude（财产规模）
- **类型**: 类别型
- **说明**: 客户拥有的财产类型
- **取值**:
  - A121: 房地产
  - A122: 建筑协会储蓄协议/人寿保险
  - A123: 汽车/其他
  - A124: 无财产/未知

### 13. age（年龄）
- **类型**: 数值型
- **说明**: 客户年龄（岁）
- **取值范围**: 19 - 75岁

### 14. other_payment_plans（其他还款计划）
- **类型**: 类别型
- **说明**: 其他分期付款计划
- **取值**:
  - A141: 银行
  - A142: 商店
  - A143: 无

### 15. housing（住房类型）
- **类型**: 类别型
- **说明**: 住房类型
- **取值**:
  - A151: 租房
  - A152: 自有住房
  - A153: 免费住房

### 16. existing_credits（现有信贷数量）
- **类型**: 数值型
- **说明**: 在本银行的现有信贷数量
- **取值范围**: 1 - 4

### 17. job（工作类型）
- **类型**: 类别型
- **说明**: 工作类型
- **取值**:
  - A171: 非技术工人 - 无固定住所
  - A172: 非技术工人 - 有固定住所
  - A173: 技术工人/职员
  - A174: 管理层/个体经营者/高素质员工

### 18. num_dependents（受抚养人数）
- **类型**: 数值型
- **说明**: 受抚养人数
- **取值范围**: 1 - 2

### 19. own_telephone（是否拥有电话）
- **类型**: 类别型
- **说明**: 是否拥有电话
- **取值**:
  - A191: 无
  - A192: 有（以客户名义注册）

### 20. foreign_worker（是否为外籍工人）
- **类型**: 类别型
- **说明**: 是否为外籍工人
- **取值**:
  - A201: 是
  - A202: 否

## 特征类型汇总

| 类型 | 特征数量 | 特征列表 |
|------|----------|----------|
| 数值型 | 7 | duration, credit_amount, installment_commitment, residence_since, age, existing_credits, num_dependents |
| 类别型 | 13 | checking_status, credit_history, purpose, savings_status, employment, personal_status, other_parties, property_magnitude, other_payment_plans, housing, job, own_telephone, foreign_worker |
| 目标变量 | 1 | class |
