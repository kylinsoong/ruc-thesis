=== 1 Logistic Regression（逻辑回归）

==== 模型简介

逻辑回归是一种经典的线性分类模型，通过 sigmoid 函数将线性回归的输出映射到 [0,1] 区间，表示样本属于正类的概率。

==== 适用场景

* 金融风控的基线模型
* 需要模型解释性的场景
* 特征与标签呈线性关系的任务

==== 运行命令

.运行 Logistic Regression 在 German Credit 数据集
[source,bash]
----
python run_baseline_experiment.py --model lr --dataset german_credit --output results/baseline_lr_results.csv
----

.运行 Logistic Regression 在 Credit Card 数据集
[source,bash]
----
python run_baseline_experiment.py --model lr --dataset credit_card --output results/baseline_lr_results.csv
----

==== 参数说明

|===
|参数 |说明 |默认值

|`--model`
|模型标识，固定为 `lr`
|必填

|`--dataset`
|数据集标识
|必填

|`--output`
|结果输出路径
|`results/baseline_results.csv`
|===

* 逻辑回归对特征尺度敏感，建议预处理时进行标准化
* 使用 `class_weight='balanced'` 自动处理类别不平衡


=== 2 Random Forest（随机森林）

==== 模型简介

随机森林是一种集成学习方法，通过构建多棵决策树并取其投票结果来做出最终预测，具有抗过拟合能力强、对缺失值鲁棒等优点。

==== 适用场景

* 特征维度较高的数据集
* 需要特征重要性分析的场景
* 处理类别不平衡问题

==== 运行命令

.运行 Random Forest 在 German Credit 数据集
[source,bash]
----
python run_baseline_experiment.py --model rf --dataset german_credit --output results/baseline_rf_results.csv
----

.运行 Random Forest 在 Credit Card 数据集
[source,bash]
----
python run_baseline_experiment.py --model rf --dataset credit_card --output results/baseline_rf_results.csv
----

==== 参数说明

|`--model`
|模型标识，固定为 `rf`
|必填

|`--dataset`
|数据集标识
|必填

|`--output`
|结果输出路径
|`results/baseline_results.csv`
|===

* 使用 `max_features='sqrt'` 限制每棵树使用的特征数
* 默认使用 `class_weight='balanced'` 处理不平衡


=== 3 XGBoost（梯度提升树）

==== 模型简介

XGBoost 是梯度提升决策树的高效实现，通过迭代训练决策树来最小化目标函数，在分类任务中表现出色。

==== 适用场景

* 需要高精度预测的场景
* 处理大规模数据集

=== 4 DeepFM（深度因子分解机）

==== 模型简介

DeepFM 结合了因子分解机（FM）和深度神经网络，能够同时学习低阶特征交互和高阶特征交互，适用于具有稀疏特征的数据。

==== 适用场景

* 需要捕获特征交互的场景
* 特征维度较高的数据
* 稀疏特征数据的分类任务

=== 5 TabNet（表数据神经网络）
